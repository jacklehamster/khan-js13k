# To play a song:


## Wild Horde
```javascript
const song = new Song(wildHorde);
song.prepare();
//  Then later...
song.play();
song.stop();
```

## Fur Elise
```javascript
const song = new Song(furElise);
song.prepare();
//  Then later...
song.play();
song.stop();
```

- soundbox.js
- song.js
- fur-elise.js / wild-horde.js
